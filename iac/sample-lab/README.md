# Sample Lab Preparation Scripts

In this directory, you will find the definition structures to prepare and destroy a sample lab environment for hackathons:

* [deploy-lab.ps1](deploy-lab.ps1): Deploys the lab environment for each team (subscription or resource group based deployment).
* [destroy-lab.ps1](destroy-lab.ps1): Destroys the lab environment for each team (subscription or resource group based deployment).
* [quota-requests.csv](quota-requests.csv): Defines the required Azure Quota resources per subscription
