# Azure Environment Preparation Helpers

In this directory, you will find useful scripts to help prepare your Azure environment for hackathons.


## Scripts

 - [createEntraIdUsers.ps1](createEntraIdUsers.ps1): A PowerShell script to create Entra ID users in your Entra ID.
 - [deleteEntraIdUsers.ps1](deleteEntraIdUsers.ps1): A PowerShell script to delete Entra ID users from your Entra ID.
